Frontend CineFlix - instruções
- Abra frontend/index.html no navegador para ver a página inicial.
- O frontend espera que o backend rode em http://localhost:3001 para signup/verify.
- Para usar envio real via SendGrid: configure backend/.env e rode o backend.
- Arquivos importantes:
  frontend/index.html, frontend/home.html, frontend/filme.html, frontend/login.html, frontend/plans.html, frontend/conta.html
